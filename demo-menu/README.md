# Menú Digital TV — Demo

Demo de menú digital para TVs en restaurantes, hecho por Teknobot.

## Características

- Auto-rotación entre categorías (12 segundos por defecto)
- Diseño full-screen optimizado para TVs 1080p / 4K
- Imágenes de alta resolución por plato
- Animaciones suaves entre categorías
- Ticker de información abajo (horarios, contacto)
- 4 categorías: Desayunos, Almuerzos, Bebidas, Postres
- Layouts variados (featured vs grid uniforme)

## Controles (modo demo)

- **Flecha izquierda/derecha** — cambiar categoría
- **Espacio** — pausar/reanudar auto-rotación
- **F** — pantalla completa
- Mover el cursor a la esquina inferior derecha para ver controles visuales

## Stack

- HTML/CSS/JS puro — sin dependencias
- Funciona en cualquier navegador moderno (Chrome, Firefox, Edge, Safari, Fire TV browser)
- Compatible con Chromecast, Fire TV Stick, Roku, Mini PC, Android TV

## Para producción

Este demo usa contenido ficticio "El Coquí Bistró". En implementación real:
- Items, precios e imágenes se personalizan para cada restaurante
- Se puede agregar panel admin para que el dueño edite sin tocar código
